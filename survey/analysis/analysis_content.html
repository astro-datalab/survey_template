Jupyter Notebook Server

There is both a public (anonymous) and private Data Lab Jupyter Notebook server to allow access and exploration of the NSC catalog and images. By clicking one of the above links, you will start an instance of this server running. You can make changes to the example notebooks, but note that these changes will disappear on the public/anonymous notebook server once you close the page or the browser. The notebooks in the private notebook server will persist across multiple sessions.
