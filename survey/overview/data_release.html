First Data Release (DR1)

The first DELVE data release (DELVE DR1) is hosted at the Astro Data Lab and consists of catalog-level coadds in griz from the DELVE-WIDE processing. DELVE DR1 covers >5,000 deg2 in each individual band and ~4,000 deg2 in all bands simultaneously. The DELVE DR1 catalog contains ~520 million unique objects.

Browse the Data Access page (side menu) to find out about different ways to use the DELVE DR1 catalog or obtain DELVE image cutouts.
