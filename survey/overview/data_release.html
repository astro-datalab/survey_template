Second Data Release (DR2)

The figure below shows the density of the 3.9 billion objects in the NSC DR2 catalog covering ~35,000 square degrees. There are 68 billion measurements from 412 thousand exposures. Access and exploration tools include the Data Lab Data Discovery tool, database access to the catalog (via direct query or TAP service), an image cutout service, and a Jupyter notebook server with example notebooks for exploratory analysis. There are five NSC DR2 tables in the database:
  • Coverage. Coverage information for each nside=4096 healpix pixel with a resolution of ~1 arcmin. This gives coverage fraction and depth for each band. 172,157,952 rows.
  • Chip. Information for each unique chip image with astrometric correction terms and uncertainties. 24,416,997 rows.
  • Exposure. Information for each unique exposure with zero-point values and uncertainties. 412,116 rows.
  • Object. Information on each unique object with average photometric and astrometric values. 3,932,838,064 rows.
  • Measurement. Information on each individual source measurement. 67,822,619,019 rows.
